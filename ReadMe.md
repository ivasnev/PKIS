# Quickstart

## Сборка окружения

1. Создайте виртуальное окружение (если еще не сделано):
   ```bash
   python -m venv venv
   ```

2. Активируйте виртуальное окружение:
   - Для Windows:
     ```bash
     venv\Scripts\activate
     ```
   - Для macOS/Linux:
     ```bash
     source venv/bin/activate
     ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
